Bienvenidos al curso programacion web 1

El principal objetivo de este curso es compartir los conocimientos básicos sobre la Vinculación de BD a una página con PHP, comenzaremos conociendo la estructura de código php y sus funciones básicas, para lograr acceder a una BD, aprenderemos como se crea y se administra una BD partiendo desde la inserción de los datos hasta lograr hacer consulta y modificaciones en la misma y mostrar el resultado en una página web, para esto conoceremos algunas herramientas que nos ayudaran en todo el proceso de aprendizaje.

Como herramienta principal para la creación de nuestro código php podremos emplear un editor de texto, como Notepad, bloc de notas, esto será más que suficiente, pero en esta ocasión utilizaremos el editor Sublime Text como editor de nuestro código fuente, por su entorno amigable y su gran versatilidad.

Para la creación de nuestra base de datos emplearemos como herramienta principal el sistema administrador de bases de datos relacionales MySQL, dado que es el gestor de código abierto más popular en la actualidad y lo encontramos disponible para distintos sistemas operativos, como lo son Windows, Linux, Mac y muchos más. Entre lo más relevante de MySQL encontramos que forma parte como sistema gestor de base de datos de las plataformas LAMP, BAMP, MAMP Y WAMP, incrementando con esto su popularidad en el desarrollo de aplicaciones web.
